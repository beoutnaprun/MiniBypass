# MiniBypass 免杀工具
=


![1](https://github.com/beoutnaprun/MiniBypass/assets/133112969/f89c44d9-afcd-46e8-b7a6-a5c8d5e2fdeb)

工具介绍：
-
    单个文件静态动态免杀
    需要python3 x64 环境运行、且不能有python2环境（因需要python命令）

使用方法
-
    直接打开文件输入shellcode 会在当前目录生成result文件夹 文件夹内是生成的远控木马文件
